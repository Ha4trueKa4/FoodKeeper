# 📸 Функциональность работы с картинками

## ✅ Что было добавлено

### 1. **ImagePickerButton.kt** - Компонент выбора изображения
Новый компонент позволяет пользователям выбирать изображения двумя способами:
- 📷 **Камера** - снять фото прямо в приложении
- 🎨 **Галерея** - выбрать фото из галереи телефона

**Особенности:**
- Красивый UI с предпросмотром выбранного фото
- AlertDialog для выбора источника
- Автоматические разрешения через ActivityResult API
- Работа с FileProvider для безопасности

### 2. **Обновлен AddProductScreen.kt**
- Добавлен ImagePickerButton в начало формы
- Фото теперь сохраняется вместе с продуктом
- URI изображения преобразуется в строку для хранения в БД

### 3. **Обновлен ProductCard.kt**
- Карточки теперь отображают загруженные фото
- Если фото нет, показывается дефолтное изображение (молоко)
- Используется Coil для асинхронной загрузки изображений

### 4. **Добавлены зависимости**
- **Coil 3** - библиотека для загрузки и кэширования изображений
- Добавлены необходимые разрешения в AndroidManifest.xml

### 5. **FileProvider конфигурация**
- Создан `file_paths.xml` для безопасной работы с файлами
- Зарегистрирован FileProvider в манифесте

## 🎨 UI Компоненты

### ImagePickerButton
```kotlin
ImagePickerButton(
    selectedImageUri = imageUri,
    onImageSelected = { uri ->
        imageUri = uri
    }
)
```

Компонент отображает:
- Кнопку "➕ Добавить фото" / "Изменить фото"
- Превью выбранного фото (200.dp height)
- AlertDialog с выбором Камера/Галерея

## 🔄 Поток использования

1. **Пользователь нажимает на кнопку "Добавить фото"**
   ↓
2. **Открывается диалог с выбором источника**
   ↓
3. **Если Галерея** - стандартный FilePicker
   **Если Камера** - запрос разрешения + интент на камеру
   ↓
4. **Выбранное изображение отображается в превью**
   ↓
5. **При сохранении продукта URI сохраняется в БД**
   ↓
6. **На карточке отображается загруженное фото**
